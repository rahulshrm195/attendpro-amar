🆕 What's New
v1.0.0
March 2026 — Initial Release
🎉 AttendPro launched for Amar Furniture!
✅ Mark attendance: 0, 0.5, 1, 1.5 (employees) · 2 (admin only)
📅 Monthly calendar view with color coding
👥 Admin can add/edit/delete staff with login credentials
💰 Advance & expense tracking with carry-forward balance
📊 Monthly salary report per employee
📩 Leave & advance request system
🔔 In-app notifications for requests & reminders
🔑 Admin password change from profile
📱 PWA — installable on Android & iOS
☁️ Firebase Firestore — data syncs across all devices
